Frequently Asked Questions

Is Launchkit a WordPress theme?

No, Launchkit is an HTML template and HTML Page Builder all-in-one. This means there are is no CMS to install and no database to manage. All content is controlled via the browser based page builder dealing purely in HTML.

Will there be a Launchkit for WordPress?

There will likely be a WordPress version of Launchkit in the near future. While we don't have an exact date - stay tuned for updates and we'll let you know when the WP version is ready.

How many landing pages can I create?

You may create as many pages as you like per single project. This means that if you plan to use Launchkit for another project, you will need to purchase an additional standard license.

Do you provide support for Launchkit?

Yes - Medium Rare provides support for Launchkit via our dedicated support forum at mediumrare.ticksy.com. This includes general item support but not additional customisation services.

Are PSD files included in the download?

While the PSD files are not included in the download package from Envato (due to file-size), you can contact us via our support forum at mediumrare.ticksy.com and we will happily provide them for you.
